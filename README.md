i made a much more simple version of this in python months ago when i noticed CCleaner was becoming worse as a tool. 
i decided to clean it up & toss it into pyinstaller to add to my very small coding portfolio lol

here's what it cleans exactly;
user temp (junk)  
win temp (junk)  
prefetch (junk)  
thumbnails (junk)  
recycle bin (junk)  
logs & tmp (APPDATA/LOCALAPPDATA)  
downloads (30 days old)  
browser cache (nothing else)  
win update (old win updates)
